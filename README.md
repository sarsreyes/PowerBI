# Retail Discount Strategy Dashboard (Power BI)

## Overview
This project presents an interactive Power BI dashboard designed to analyze the impact of discount strategies on retail performance.

The analysis combines predictive and prescriptive analytics to support data-driven decision-making.

## Objectives
- Evaluate how discount levels affect:
  - Total Sales
  - Profit Margin
  - Total Profit
- Identify trade-offs between revenue growth and profitability
- Provide actionable insights for pricing strategies

## Tools & Technologies
- Power BI
- DAX
- Data Modeling (Star Schema)
- Forecasting (Time Series)
- Scenario Simulation (What-if Analysis)

## Key Features

### Predictive Analytics
- Sales forecasting using time series analysis
- Discount vs performance analysis (Sales, Profit, Margin)

### Prescriptive Analytics
- What-if parameter for discount simulation
- Scenario-based KPI comparison
- Real-time impact analysis

### Dashboard Insights
- KPI tracking (Sales, Profit, Margin)
- Scenario comparison metrics
- Clustering & segmentation
- Interactive slicers and filters

## Key Findings
- Discount increases do not always lead to proportional sales growth
- Higher discounts consistently reduce profit margins
- A balanced discount strategy is recommended
- Product segmentation enables more targeted pricing strategies

## Files Included
- Power BI Dashboard (.pbix)
- Final Report (PDF)
- Supporting documentation

## Notes
The dataset and simulation model are simplified for academic purposes. Results should be interpreted as analytical insights rather than exact predictions.
